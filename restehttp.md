# O que é HTTP? 

A sigla HTTP significa *Hyper Text Transfer Protocol* *, é um procotocolo de comunicação para transferencia de Hyper Text. Sendo o Hyper Text mais comum, é o HTML, *Hyper Text Markup language .* 

Ao clicar em um link, estamos chamando o método GET(ler) do HTTP.

Quando enviamos um formulário, estamos chamando o método POST (inserir).

# URIs

URIs (Identificador uniforme de recursos), são sequencias de caracteres que servem para identificar e nomear recursos online de forma unica e global. São utilizadas para localizar e acessar resucrsos, como documentos, páginas web, imagens, arquivos, serviços web... 

Os tipos principais de URIs são: 

- URL - é usada para descrever a localização exata de um recurso na Internet. As URLs geralmente incluem informações sobre o protocolo de acesso (como HTTP ou FTP), o domínio ou endereço IP do servidor e o caminho para o recurso específico.
- URN - RNs são projetados para durar a longo prazo e não indicam diretamente como acessar o recurso, ao contrário das URLs. Exemplo de URN: urn:isbn:0451450523 (para identificar um livro por seu ISBN)
